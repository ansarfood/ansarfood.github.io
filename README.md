# ANSARFOOD — лендинг франшизы

Одностраничный лендинг для франшизы **ANSARFOOD** (халал-фастфуд, Кыргызстан).
Текст на кыргызском языке, по ТЗ `ТЗ лендинг Ансар.docx`.

## Файлы

| Путь | Назначение |
|------|------------|
| `index.html` | Разметка всех блоков |
| `styles.css` | Стили, фирменная палитра, адаптив, анимации |
| `script.js`  | Калькулятор прибыли, отправка заявки в WhatsApp, scroll-анимации |
| `fonts/`     | Шрифты Geologica + Comfortaa (латиница + кириллица), захостены локально |
| `assets/logo/` | Логотип из брендбука: `logo-color.png` (для светлого) и `logo-white.png` (для тёмного/красного) |
| `assets/menu/` | 10 фото блюд из меню (webp, с прозрачным фоном) |

## Фирменный стиль (из брендбука)

- **Цвета:** красный `#d20a11`, оранжевый `#ec6608`, янтарный `#f7a600`, жёлтый `#ffdd00`, чёрный `#000`
- **Шрифты:** Geologica (текст), Comfortaa (заголовки — замена имиджевого AQUM)
- **Логотип:** оригинальный знак-пламя «ANSAR food», извлечён из `Ansar brandbook.pdf`

## Блоки

1. Hero — слоган, оффер, ключевые цифры
2. Артыкчылыктар — 4 преимущества
3. Бренддин жүрөгү — история + цитата основателя
4. Кимдер үчүн — для кого
5. **Меню** — 10 позиций (бургеры, крылышки, шаурма/шаверма)
6. Эки формат — Фудтрак / Кафе
7. Пайда калькулятору — интерактивный расчёт
8. Жол картасы — 6 шагов
9. Байланыш — форма заявки (WhatsApp) + контакты

## WhatsApp

Форма заявки **открывает WhatsApp** с уже заполненным сообщением (имя, телефон, город, формат)
на номер `+996 703 512 044`. Также есть зелёная плавающая кнопка WhatsApp и ссылки в контактах/подвале.

**Сменить номер:** в `script.js` измените переменную `WA_PHONE` (формат: только цифры, напр. `996703512044`).
Также обновите ссылки `https://wa.me/996703512044` в `index.html` (контакты, подвал, кнопка-«фаб»).

## Запуск локально

```bash
cd ansar
python3 -m http.server 8123
# открыть http://localhost:8123
```

## Деплой

Статический сайт — Netlify / Vercel / GitHub Pages или любой хостинг: загрузите все файлы и папки в корень.

## Замена фото меню

Картинки лежат в `assets/menu/*.webp` (прозрачный фон). Просто замените файл, сохранив имя,
или поменяйте `src`/названия в блоке `<section id="menu">` в `index.html`.
